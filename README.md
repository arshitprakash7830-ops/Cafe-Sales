# Cafe-Sales
Developed an interactive Cafe Sales Dashboard in Excel analyzing 73K+ transactions. Tracked sales, revenue, top products, payment methods, and order trends. Built dynamic charts and slicers for day/month filtering, revealing peak sales on Fridays and Sundays while enabling data-driven inventory and operational decisions.
